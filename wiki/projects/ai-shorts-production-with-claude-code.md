# Claude Code로 AI 쇼츠 영상 대량 제작

Claude Code는 코드 작성뿐 아니라 반복 콘텐츠 제작 파이프라인을 설계하고 실행하는 데도 사용할 수 있다. 쇼츠 영상 제작에서는 주제 선정, 스크립트 생성, 제목/업로드 일정, 댓글 템플릿까지 일괄 생성하는 식으로 활용한다.

## 핵심 내용

### 전체 흐름

1. 채널 콘셉트와 타깃 시청자를 정한다.
2. 바이럴 사례를 분석한다.
3. Claude Code로 짧은 스크립트를 여러 편 생성한다.
4. 영상 제작 도구로 나레이션, 자막, 이미지/영상 소스를 결합한다.
5. 제목, 설명, 해시태그, 업로드 일정을 만든다.
6. 조회수와 댓글 반응을 보고 다음 배치를 개선한다.

### Claude Code가 맡기 좋은 부분

- 1개월치 주제 리스트 생성
- 쇼츠용 30-60초 스크립트 대량 생성
- 바이럴 제목 후보 생성
- 댓글 답변 템플릿 작성
- 업로드 캘린더 작성
- 조회수 부진 원인 가설 정리

### 사람이 검토해야 하는 부분

- 사실관계
- 저작권
- 플랫폼 정책
- 과장 광고 여부
- 목소리·영상 품질
- 브랜드 톤

## 세부 사항

### 추천 프롬프트 구조

```markdown
목표: 30초 쇼츠 영상 스크립트 10개 생성
대상: [타깃 시청자]
톤: [친근함/전문적/자극적이지 않음]
형식:
- 훅 1문장
- 본문 3문장
- 마무리 1문장
금지:
- 확인되지 않은 수치
- 과장된 수익 표현
- 저작권 침해 가능 문구
```

### 자주 하는 실수

| 실수 | 대응 |
|---|---|
| 아이디어가 떠오르지 않음 | 채널 콘셉트와 타깃을 먼저 좁힘 |
| 지속이 안 됨 | 1개월치 주제와 업로드 일정을 먼저 생성 |
| 조회수가 늘지 않음 | 훅, 제목, 썸네일, 첫 3초를 분리해 개선 |
| 품질이 균일하지 않음 | 스크립트 템플릿과 검수 체크리스트 고정 |

### 강의 녹취 메모

녹취에서는 무료 TTS 도구로 Clova를 언급했지만, 품질 면에서는 해외 솔루션이 더 나을 수 있다고 설명했다. 무료 도구는 시작 비용이 낮다는 장점이 있고, 본격 운영 단계에서는 음성 품질과 제작 속도를 함께 비교해야 한다.
