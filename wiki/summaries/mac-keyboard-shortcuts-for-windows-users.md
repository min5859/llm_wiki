# 맥 키보드 단축키 정리 (윈도우 사용자용)

맥은 윈도우와 가장 큰 차이가 **`Ctrl` 대신 `Command(⌘)` 키를 많이 쓴다**는 점이다. 처음엔 꽤 헷갈리지만, 자주 쓰는 것만 익히면 금방 적응할 수 있다.

## 가장 먼저 익힐 핵심

### 복사 / 붙여넣기 / 잘라내기
- **복사**: `⌘ + C`
- **붙여넣기**: `⌘ + V`
- **잘라내기(텍스트)**: `⌘ + X`
- **실행 취소**: `⌘ + Z`
- **다시 실행**: `Shift + ⌘ + Z`

윈도우의 `Ctrl + C / V / X`는 맥에서 거의 그대로 **`Command + C / V / X`** 로 생각하면 된다.

## 스크린샷
맥에서는 아래 3개를 알면 거의 충분하다.

- **전체 화면 캡처**: `Shift + ⌘ + 3`
- **영역 선택 캡처**: `Shift + ⌘ + 4`
- **캡처 도구 열기**: `Shift + ⌘ + 5`

### 참고
- 윈도우의 `Shift + Win + S`에 가장 가까운 것은 **`Shift + ⌘ + 5`**
- `Shift + ⌘ + 4` 후 `Space`를 누르면 **창 단위 캡처** 가능

## 선택 / 찾기 / 저장
- **전체 선택**: `⌘ + A`
- **찾기**: `⌘ + F`
- **저장**: `⌘ + S`
- **열기**: `⌘ + O`
- **새 문서/새 창**: `⌘ + N`
- **인쇄**: `⌘ + P`

## 창 / 앱 관리
- **창 닫기**: `⌘ + W`
- **앱 종료**: `⌘ + Q`
- **최소화**: `⌘ + M`
- **앱 전환**: `⌘ + Tab`
- **같은 앱 내 창 전환**: ``⌘ + ` ``
- **현재 앱 숨기기**: `⌘ + H`

### 중요한 차이
- `⌘ + W` = **창만 닫기**
- `⌘ + Q` = **앱 완전 종료**

윈도우처럼 창을 닫으면 앱이 끝난다고 생각하면 헷갈리기 쉽다.

## 브라우저 / 문서 작업에서 자주 쓰는 것
- **새 탭**: `⌘ + T`
- **탭 닫기**: `⌘ + W`
- **닫은 탭 다시 열기**: `Shift + ⌘ + T`
- **새로고침**: `⌘ + R`
- **주소창 이동**: `⌘ + L`

## 커서 이동 / 편집
- **줄 처음**: `⌘ + ←`
- **줄 끝**: `⌘ + →`
- **문서 맨 위**: `⌘ + ↑`
- **문서 맨 아래**: `⌘ + ↓`
- **단어 단위 이동**: `Option + ← / →`

## 삭제 키
맥에서는 `Delete` 키가 윈도우의 Backspace 역할에 가깝다.

- **앞 글자 삭제 (Backspace)**: `Delete`
- **뒤 글자 삭제 (윈도우 Delete 느낌)**: `Fn + Delete`

## Page Up / Down / Home / End
맥 키보드에는 전용 키가 없는 경우가 많아서 보통 `Fn` 조합을 사용한다.

- **Page Up**: `Fn + ↑`
- **Page Down**: `Fn + ↓`
- **Home**: `Fn + ←`
- **End**: `Fn + →`

## Finder(파일 관리)
- **새 Finder 창**: `⌘ + N`
- **새 폴더**: `Shift + ⌘ + N`
- **파일 삭제(휴지통으로)**: `⌘ + Delete`
- **이름 바꾸기**: `Enter`

## 터미널에서 주의할 점
일반 앱에서는 복사/붙여넣기가 `⌘ + C / ⌘ + V`지만, 터미널에서는 `Ctrl + C`가 복사가 아니라 **현재 실행 중인 작업 중단**이다.

- 일반 앱 복사: `⌘ + C`
- 일반 앱 붙여넣기: `⌘ + V`
- 터미널에서 `Ctrl + C`: **프로세스 중단**

## 윈도우 ↔ 맥 대응표
- `Ctrl + C` → `⌘ + C`
- `Ctrl + V` → `⌘ + V`
- `Ctrl + X` → `⌘ + X`
- `Ctrl + Z` → `⌘ + Z`
- `Alt + Tab` → `⌘ + Tab`
- `Alt + F4` → `⌘ + Q`
- `Shift + Win + S` → `Shift + ⌘ + 5`
- `Page Up / Down` → `Fn + ↑ / ↓`
- `Home / End` → `Fn + ← / →`

## 가장 먼저 외우면 좋은 10개
1. `⌘ + C` 복사
2. `⌘ + V` 붙여넣기
3. `⌘ + Z` 실행 취소
4. `⌘ + S` 저장
5. `⌘ + F` 찾기
6. `⌘ + W` 창 닫기
7. `⌘ + Q` 앱 종료
8. `⌘ + Tab` 앱 전환
9. `Shift + ⌘ + 5` 스크린샷
10. `Fn + ↑ / ↓` 페이지 업다운

## 나의 적용 포인트
- 윈도우에서 쓰던 `Ctrl` 중심 습관을 맥에서는 `⌘` 중심으로 바꿔야 한다.
- 스크린샷은 `Shift + ⌘ + 5` 하나만 먼저 익혀도 실사용에 충분하다.
- `Fn + ↑ / ↓`, `Fn + ← / →`를 익히면 Page Up/Down, Home/End 불편이 크게 줄어든다.
- 맥에서는 창 닫기(`⌘ + W`)와 앱 종료(`⌘ + Q`)를 구분해서 기억해야 한다.
