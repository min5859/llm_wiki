# Agentic World Modeling — L1/L2/L3 역량 분류체계와 4가지 지배 법칙 체제

## 한줄 요약

AI 에이전트가 환경을 이해하고 예측하는 능력을 **L1 예측기 → L2 시뮬레이터 → L3 진화자**의 3단계 역량 분류체계로 체계화한 400편 이상의 문헌을 망라한 대규모 서베이 논문.

HKUST·NUS·Oxford·NTU·CUHK 등 다기관 공동 저술 (arXiv 2604.22748, 2026).

## 연구 배경

AI 시스템이 텍스트 생성에서 목표 달성형 지속 상호작용으로 진화하면서, **세계 모델(World Model)**이 핵심 병목으로 부상했다. 그러나 "세계 모델"이라는 용어는 분야마다 다르게 사용되어 왔다:

- **RL**: 전이 구조를 학습해 행동 전에 미래를 상상
- **컴퓨터 비전**: 시각적 일관성을 유지하는 비디오/3D 생성기
- **언어 모델**: 계획·웹 상호작용을 위한 텍스트 기반 시뮬레이션
- **로보틱스**: 안전한 계획과 sim-to-real 전달을 위한 학습 역학

기존 서베이들은 **양식(modality) 또는 응용 도메인** 중심으로 분류하여 분야 간 단절을 심화시켰다. 본 논문은 이를 해소하고 **역량 기반 분류체계**를 공통 언어로 제시한다.

## 핵심 분류체계

### "레벨 × 법칙" 2축 구조

#### 역량 레벨 (3단계)

| 레벨 | 이름 | 핵심 능력 | 철학적 유비 |
|------|------|-----------|------------|
| **L1** | Predictor | 단일 스텝 로컬 전이 연산자 학습 | 흄의 '상수 연접' — 패턴을 학습하나 인과성 미보장 |
| **L2** | Simulator | L1 연산자를 다단계 행동 조건부 롤아웃으로 합성, 도메인 법칙 준수 | 루이스의 '가장 가까운 가능 세계' — 개입 조건부 반사실 추론 |
| **L3** | Evolver | 예측 실패 시 모델 구조 자체를 자율 수정 | 라카토스의 '핵(hard core) vs 보호대(protective belt)' — 증거 기반 모델 개정 |

L1→L2의 경계는 **단일 스텝 예측 품질이 아니라 합성 하의 롤아웃 충실도**로 결정된다.

**L2 경계 조건 3가지:**
1. **장기 일관성**: 롤아웃이 여러 스텝에 걸쳐 유용하게 유지
2. **개입 민감성**: 행동/전제 변경이 안정적이고 의미 있는 궤적 변화를 유발
3. **제약 일관성**: 생성된 미래가 대상 체제의 지배 법칙 제약을 준수

**L3 경계:** "모델 자체가 개정의 대상"이 됨. L3는 L2를 대체하지 않고, L2가 체계적으로 실패할 때 발동하는 거버넌스 레이어.

#### 수식 표현 (POMDP 기반)

```
L1: pθ(zt | zt-1, at)                          — 단일 스텝 잠재 전이
L2: p̂(τ | z₀, a₁:H, c) ∝ ∏pθ(zt | zt-1, at) · φc(τ)  — 제약 φc 포함 다단계 롤아웃
L3: Mt → Mt+1  (evidence-driven model revision)  — 모델 스택 자체 개정
```

#### 지배 법칙 체제 (4가지)

| 체제 | 제약 조건 | 검증 방법 | 위반 예시 |
|------|----------|----------|----------|
| **물리 세계** | 접촉역학·중력·운동학적 실현 가능성 | 해석적/수치적 기준해 | 물체가 테이블을 통과 |
| **디지털 세계** | API 계약·UI 상태 기계·프로그램 의미론 (결정론적) | 실행 및 출력 비교 | 존재하지 않는 API 호출 |
| **사회 세계** | 믿음·목표·규범 (반성적·규범적) | 상호 기대 정합성 | 약속 이행 없이 무시 |
| **과학 세계** | 잠재 인과 메커니즘 (실험적으로 발견) | 실험실/관측 데이터 비교 | 보존 법칙 위반 분자 구성 예측 |

## 대표 시스템 및 주요 결과

- **TD-MPC2**: 단일 317M 파라미터 에이전트로 104개 과제 마스터
- **EfficientZero**: Atari에서 **2시간** 경험으로 초인간 성능
- **AlphaFold 계열**: L1 상태 추론 패러다임의 과학 세계 적용 — 거의 실험적 정확도
- **MuZero**: 관측 재구성 없이 가치 예측+MCTS를 결합한 결정론적 잠재 동역학
- **Dreamer 계열(V1→V3)**: RSSM 기반 잠재 상상 강화학습

## 한계점

- **L3 미성숙**: 대부분 시스템은 L1·L2에 해당, L3는 부분적 구현 단계
- **표현 형식 문제**: 신경망 잠재 표현은 L3의 법칙 구조 수정에 부적합 — **기호(symbolic) 기반 표현** 필요할 수 있음
- **복합 체제 평가 기준 부재**: 자율주행처럼 물리+사회 세계가 동시에 적용되는 케이스
- **평가 표준화**: 커뮤니티마다 지표가 달라 비교 어려움

## 실무 적용 포인트

- **로보틱스/자율주행**: L2 시뮬레이터로 배포 전 가상 실험 → 안전성·데이터 효율 향상
- **소프트웨어 에이전트(GUI/웹 자동화)**: 디지털 세계 체제의 L2로 복잡한 UI 탐색·API 상호작용 계획
- **AI 기반 과학 발견**: 가설 주도 실험 자동화 (L3 방향)
- **합성 데이터 생성**: L2 시뮬레이터가 에이전트 훈련용 현실적 경험 데이터를 자동 생성
