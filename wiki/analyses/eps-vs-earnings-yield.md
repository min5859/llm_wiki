# EPS vs Earnings Yield (EY)

종목 스코어링에서 **이익을 점수화할 때 절대값(EPS)을 쓰면 안 되고 비율(EY)을 써야 하는 이유**. "절대 금액을 점수 차원으로 직접 쓰면 스케일이 큰 쪽이 부당하게 유리해진다"는 일반 원리의 구체 사례로, 자동매매·재무 대시보드의 채점 함수 설계에 반복 등장한다.

## 정의

- **EPS (Earnings Per Share, 주당순이익)** — 1주가 1년에 벌어들인 순이익. **절대 금액(원)**.
  - 예: 삼성전자 EPS 5,700원.
- **EY (Earnings Yield, 이익수익률)** — EPS를 주가로 나눈 **비율(%)**. `EPS / 주가 × 100`. PER의 역수.
  - 예: 5,700 / 70,000 × 100 = **8.1%**.

## 왜 EPS 절대값으로 채점하면 안 되는가

EPS 절대값에 점수 구간을 매기면 **주가가 비쌀수록 무조건 유리해지는 편향**이 생긴다. 같은 EPS 5,700원이라도:

| 주가 | EPS | EY | 실제 밸류 | EPS 절대 채점 | EY 채점 |
|---|---|---|---|---|---|
| 70,000원 | 5,700 | 8.1% | 저평가 | (동일 점수) | 높음 |
| 500,000원 | 5,700 | 1.1% | 고평가 | (동일 점수) | 낮음 |

EPS 절대값만 보면 두 종목이 같은 점수를 받지만, 실제로는 7만원 주식이 8% 수익률로 훨씬 싸다. EY는 "이 가격에 사면 연 몇 % 버는 셈인가"를 나타내 **종목 간 비교가 가능**하다.

## 채점 구간 기준

KOSPI 평균 EY가 5~8% 대이므로 채점 구간을 이 기준에 맞춘다 (예시):

```
EY ≥ 12% → 10점   (강한 저평가)
EY ≥  8% →  8점
EY ≥  5% →  5점
EY ≥  3% →  2점
```

## 일반화 — "절대값을 점수 차원으로 직접 쓰지 마라"

이 사례의 일반 원리는 **스케일이 다른 절대값을 점수 차원에 직접 매핑하지 말고, 비교 가능한 비율/정규화 형태로 바꿔라**는 것이다. 같은 함정이 반복되는 곳:

- 시가총액 절대값 대신 거래대금 회전율
- 거래량 절대값 대신 20일 평균 대비 배율
- 매출 절대값 대신 성장률/마진

PER 자체도 같은 EY의 변형(역수)이지만, EY는 0 근처·음수 처리(적자)가 PER보다 직관적이라 점수 함수에서 다루기 쉽다. 단 PER/EY 모두 섹터마다 정상 범위가 달라, 절대 구간 대신 **섹터 상대화**가 다음 개선이 된다 ([[stock-screening-score-design]]).
