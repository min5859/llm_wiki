# LLaDA2.0-Uni — 이산 확산 LLM 기반 멀티모달 통합 모델

## 한줄 요약

완전 의미론적 이산 토크나이저(SigLIP-VQ)와 MoE dLLM 백본, 확산 디코더를 통합해 이해·생성·편집을 단일 마스크 예측 목표로 학습하는 멀티모달 프레임워크.

## 연구 배경

기존 통합 멀티모달 모델의 두 가지 흐름:
- **AR 기반** (Janus, BAGEL 등): 이미지 이산화 + 다음 토큰 예측. 텍스트와 이미지를 별도 인코더로 처리해 최적화 목표가 분리됨.
- **마스크 확산 기반** (MMaDA, Lumina-DiMOO 등): 병렬 디코딩과 양방향 컨텍스트 모델링 장점이 있으나, 재구성 기반 VQ 토크나이저를 사용해 의미 정보가 부족하고 이해 성능이 낮음.

LLaDA2.0-Uni는 **"이해와 생성 모두에 완전 이산 의미론적 토큰 사용"** 원칙으로 이 문제를 해결한다.

## 핵심 방법론

### 3계층 아키텍처

**1. SigLIP-VQ 토크나이저 (의미론적 이산화)**
- SigLIP2-g ViT를 시각 특징 추출기로 사용
- 픽셀 재구성이 아닌 **이해 태스크 직접 학습** → 의미 정보 보존
- 코드북: 어휘 크기 16,384, 차원 2,048
- 동적 해상도 지원 (`<height>`, `<width>` 특수 토큰으로 1D RoPE 유지)

**2. 16B MoE dLLM 백본 (LLaDA2.0-mini 기반)**
- 텍스트+이미지 토큰을 동일한 블록 단위 마스크 예측 목표로 처리
- **Block-wise Attention**: 완전 양방향 어텐션 대신 블록 내 어텐션 + 선택적 블록 간 어텐션 → 학습 안정성 확보. SigLIP-VQ 토큰이 Qwen2.5와 정렬되어 있어 AR 편향을 유지할 필요 있음.
- 어휘 확장: 원래 언어 임베딩 유지 + SigLIP-VQ 코드북 토큰 추가

**3. 확산 디코더 (Z-Image-Base 6B 기반)**
- dLLM이 생성한 이산 토큰을 조건부 신호로 사용 (기존 방법들의 텍스트+시각 토큰 중복 조건 없음)
- 2× 수퍼해상도 포함
- 50단계 샘플링 → **지식 증류로 8단계 CFG-free 추론** 달성

### SPRINT 추론 가속 (학습-free)

블록 단위 dLLM은 *B×T* 포워드 패스 필요 → 두 축으로 비용 절감:

1. **Sparse Prefix Retention**: 각 블록 첫 스텝에서 전체 KV 캐시 계산 후 모달리티별 중요도 점수로 프루닝. 이미지 토큰(공간 중복 높음)은 공격적으로, 텍스트 토큰은 보수적으로 유지. `r_text = 1.0, r_img = 0.8`
2. **Non-uniform Token Unmasking**: 고정 스케줄 대신 신뢰도(confidence) 적응형 언마스킹. `τ ∈ {0.93, 0.95}` 임계값 초과 시 해당 포지션 즉시 수락.
- 합산 효과: **품질 손실 없이 최대 1.6× 속도 향상**

## 학습 파이프라인

| 단계 | 데이터 규모 | 목적 |
|------|------------|------|
| 이해 사전학습 | 대규모 이미지-캡션 | OCR, 공간, 세계 지식 |
| 이해 SFT | ~6천만 샘플 (텍스트:멀티모달 = 1:5) | VQA, 차트, 수학적 추론 |
| 생성 데이터 | 2억 → 1.4억 (필터링 후) | 미적 점수, DeQA-Score ≥ 4.0 |
| 편집 데이터 | 오픈소스 + 합성 쌍 | 편집-생성 일관성 |
| 인터리빙 데이터 | Koala36M 비디오 코퍼스 | 이미지-텍스트 교차 생성 |

캡션은 Qwen3-VL-235B-22B가 원본 웹 텍스트 활용 여부를 평가해 고유명사 등 실제 지식 보존.

## 주요 결과

- 멀티모달 이해: 특화 VLM(Qwen2.5-VL)과 경쟁적
- 이미지 생성·편집: 통합 모델 중 상위권
- 인터리빙 생성·추론: 단일 모델로 지원 (대부분 통합 모델은 미지원)
- SPRINT: 1.6× 속도 향상, 품질 손실 미미

## 한계 및 향후 과제

- 완전 양방향 어텐션 vs Block-wise 어텐션: 이론적으로는 완전 양방향이 이상적이나 실제로는 품질 저하 → 이 갭 해소 필요
- SigLIP-VQ의 의미 보존 vs 재구성 품질 트레이드오프 (디코더로 보완하지만 추가 파라미터 비용)
- 3B 이하 소형 모델 확장성 미검증

## 설계 인사이트

- **핵심 구분선**: 재구성 기반 VQ(픽셀 재구성으로 학습) vs 의미 기반 VQ(이해 태스크로 학습). 후자가 멀티모달 이해에 결정적으로 유리.
- **Block-wise Attention 필요성**: SigLIP-VQ 토큰이 LLM과 의미 정렬되어 있어 순수 양방향 어텐션 적용 시 AR 편향 손실. 하이브리드 어텐션으로 절충.
- **SPRINT의 모달리티 인식 프루닝**: 이미지와 텍스트의 정보 밀도 차이를 활용한 비대칭 KV 캐시 전략.

## 관련 논문

- **MMaDA / Lumina-DiMOO** — 재구성 VQ 기반 마스크 확산 통합 모델 (이 연구가 극복하려는 선행 작업)
