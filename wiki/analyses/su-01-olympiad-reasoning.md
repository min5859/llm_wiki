# SU-01: Achieving Gold-Medal-Level Olympiad Reasoning via Simple and Unified Scaling

## 한 줄 요약

30B-A3B (활성 파라미터 3B) 소형 모델이 「역퍼플렉시티 커리큘럼 SFT + 2단계 RL + 테스트 타임 스케일링」 의 단순·통합 레시피만으로 IMO 2025 / USAMO 2026 금메달 수준 (35점 만점에서 동점 1위 35점) 의 수학·과학 올림피아드 성능을 달성. 저자: 상하이 AI Lab + 칭화 + CUHK 등. arXiv 2605.13301.

## 개요

이 논문은 「범용 포스트 트레이닝 모델을 *간결하고 도메인 통합된* 레시피로 올림피아드 수준까지 끌어올릴 수 있는가?」 라는 질문에 답하기 위해, 30B 규모의 베이스 모델 위에 4단계 학습/추론 파이프라인을 적용한다. AlphaProof·Gemini Deep Think 등 선행 시스템이 금메달 수준에 도달했으나 대부분 대규모 탐색 예산이나 특수 목적 아키텍처에 의존했던 것과 달리, SU-01 은 SFT + RL 의 표준 파이프라인을 *재배열* 하는 것만으로 같은 결과를 낸다.

## 4단계 레시피

### 1. 역퍼플렉시티(Reverse-Perplexity) 커리큘럼 SFT

- 338K 개 trajectory (각 8K 토큰 이하) 를, 정책 모델이 가장 *생소하게* 느끼는 고-PPL 예제부터 시작해 저-PPL 순서로 학습
- 데이터 구성: 직접 생성 (수학/STEM/코딩/IF) 54.3% + 자기 개선 (Self-Verify / Self-Refine) 45.7%
- 의도: 모델이 *어렵게* 느끼는 것부터 학습해 증명 탐색·자기 검증 행동을 *기존 능력 손상 없이* 주입

### 2. Coarse RL (거친 강화학습)

- 8,967 개 검증 가능한 프롬프트 (정답 정합성 자동 채점)
- **GSPO** (Group Sequence Policy Optimization) 알고리즘 + 이진 결과 보상
- 목적: 탐색 범위 확대 + 직접 풀이 능력 향상

### 3. Refined RL (정밀 강화학습)

- 정답 일치가 아닌 *완전한 증명 품질* 을 채점 — **DeepSeekMath-V2 기반 생성형 보상 모델** 사용
- 두 종류의 경험 버퍼:
  - **자기 수정 (Self-Refinement) 버퍼** (비율 20%) — 실패 사례를 수정 과제로 재활용
  - **경험 재생 (Experience Replay) 버퍼** (비율 25%) — 어려운 문제에서 드물게 성공한 trajectory 를 보존해 재학습 ([[ExGRPO]] 의 직접 영향)

### 4. 테스트 타임 스케일링 (TTS)

- 풀이 → 검증 → 수정 루프, 최대 30 사이클 × 10회 독립 실행
- 5회 연속 검증 통과 시 채택, 10회 연속 실패 시 종료
- 단일 trajectory 100K+ 토큰 활용

## 주요 결과

| 벤치마크 | SU-01 직접 생성 | SU-01 + TTS |
|---|---:|---:|
| IMO-ProofBench Overall | 57.6% | **70.2%** |
| IMO-ProofBench Basic | 77.1% | 91.0% |
| IMO-ProofBench Advanced | 38.1% | 49.5% |
| AIME 2025 / 2026 | 94.6% / 93.3% | — |
| IPhO 2024 (금메달 기준 20.8점) | 23.5점 | 25.3점 |
| IPhO 2025 (금메달 기준 19.7점) | 20.3점 | 21.7점 |
| IMO 2025 합계 (금메달 기준 35점) | 21점 | **35점** |
| USAMO 2026 합계 (금메달 기준 25점) | 15점 | **35점** |

특히 **USAMO 2026 에서 TTS 적용 시 340 명 참가자 중 최고 점수 (35점) 와 동점**. FrontierScience-Research 에서도 유사 크기 모델 중 최고 (11.7%) 를 달성해 물리·화학 영역으로의 전이 가능성을 시사.

## 한계점

- **IMO 2025 P6** (조합론적 구조 불변성 증명) / **USAMO 2026 P2** (글로벌 전략 논증) 에서 완전 풀이 실패 — 좌표·모듈러 산술 등 *형식적 표현으로 환원하기 어려운* 문제에 취약
- 보상 모델 (DeepSeekMath-V2) 의 노이즈 / *검증 해킹* 가능성 잔존
- TTS 비용이 상당 — 실배포 시 추론 예산 관리 필요
- 생물학 등 RL 학습 신호가 없는 도메인에서는 25% 수준에 머무름

## 실무 적용 가능성

- **교육 / 자동 채점** — 증명 품질을 평가하는 *생성형 보상 모델* 패턴은 수학 교육 플랫폼의 논술형 풀이 채점에 즉시 활용 가능
- **과학 연구 보조** — FrontierScience-Research 의 성적으로 봐 화학·물리 분야 다단계 추론 보조 도구로 확장 가능
- **소형 모델 특화** — 활성 파라미터 3B 로 GPT-5 수준 추론을 달성한 점이 *온프레미스 배포 비용 절감* 에 직접 기여
- **자동 증명 검증기** — 자기 검증-수정 루프 구조 자체를 독립된 형식 검증 파이프라인으로 재사용 가능

## 관련 논문

1. **DeepSeek-R1** (Guo et al., 2025) — 대규모 RLVR 로 장문 추론 + 자기 수정 행동을 유도한 선행 연구. SU-01 의 RL 설계 기반
2. **ExGRPO** (Zhan et al., 2025) — 경험 재생을 통해 희귀 성공 trajectory 를 재사용하는 기법. SU-01 의 경험 재생 모듈의 직접 원형
3. **Winning Gold at IMO 2025** (Huang & Yang, 2025) — 모델 비종속 검증-수정 파이프라인으로 IMO 금메달 달성한 TTS 기법. SU-01 의 TTS 설계의 직접 참조

## 일반 교훈

- **표준 컴포넌트의 *재배열* 만으로 SOTA 가능** — 신규 아키텍처 / 새 알고리즘 없이 SFT 커리큘럼 순서 + RL 2 단계 + TTS 루프의 *조합 디자인* 이 30B 모델을 IMO 금메달 라인에 올림. *컴포넌트 발명* 보다 *컴포넌트 결합* 이 ROI 가 클 수 있음을 시사
- **자기 수정 버퍼 + 경험 재생 버퍼 의 비율 디자인** — 20% / 25% 라는 명시적 ratio 가 RL 학습 안정성에 중요. 일반화 시에도 *실패-수정* 과 *성공-재생* 두 종류 신호의 균형이 후속 RL 디자인의 hyperparameter 후보
