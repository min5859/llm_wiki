# AI 요약

## 인사이트·노하우

- **Claude Max 플랜 토큰 소진 이슈**: Claude Max 사용자 중 채팅 두 번 만에 5시간 세션 한도를 소진하는 현상 발생 → 운영자가 "버그"로 확인. 정상 사용 시에도 Opus 4.8 사고 모드(thinking) 켜고 기술적 질문 3~4회 하면 용량 소진됨.
  - "보통 빡쌘거 시키면 채팅한번에 10%정도 먹던데 50퍼라니 엄청나군요" → 심층채팅(브레인스토밍/research depth 깊게)이 원인으로 추정
  - 완화책: depth를 2~3으로 줄이면 토큰 절약 가능

- **Claude 사이버보안 토픽 자동 중단**: 기술적·연구적 내용을 진행하다 보면 사이버 보안 토픽으로 분류되어 자동 중단됨. 깊이 있는 연구 작업 시 걸림돌.

- **GLM 5.2 제약사항**:
  - 구독형 사용 시 응답이 느려 타임아웃 잦음
  - 멀티모달(이미지 인식) 미지원
  - 코딩 성능은 양호하다는 평
  - 이미지 처리 필요 시 Gemini 등과 병행 사용 권장

- **OpenClaw + 슬랙 비호환**: OpenClaw가 슬랙 설정 관련 대화 중 멈추는 현상 → Hermes가 복구 수단으로 활용됨

- **Hermes 데스크탑 프로필 기능**: 프로그램 하나에 봇 토큰 여러 개를 붙여 여러 모델을 동시 운용 가능. 텔레그램도 각자 대화창 따로 사용. → 멀티 모델 운용의 실용적 방법으로 확인됨.

- **커뮤니티 링크 공유 규칙**: 홍보성과 정보성 경계의 링크는 "맥락 한두 줄"을 반드시 붙여야 함. 맥락 없이 띡 공유만 하면 홍보로 오인될 수 있음.
