## 인사이트·노하우

- **오픈클로 + 클로드코드 + 슬랙 조합 세션 만료 문제**: 스레드로 대화 중 세션이 풀리는 현상 발생. API 사용으로 전환하는 것이 해결책으로 제시됨 (미확인 — 아래 미해결 참고).
- **에르메스 슬랙 출력이 지저분한 문제 (스킬뷰·진행상황이 댓글로 모두 달림)**:
  - 해결책 1: 보기 싫은 내용을 복사해서 "그 채널에 안 보이게 해줘"라고 봇에게 요청.
  - 해결책 2: `verbose` 설정이 있으면 꺼달라고 요청 — 매번 손볼 필요 없이 일회성 설정으로 정리 가능.
  - 팁: "verbose가 뭐야? 알려줘"라고 봇에게 물어보면 개념도 배우고 설정도 바꿀 수 있어 일석이조.
- **에르메스 vs 오픈클로 특성 비교**: "에르메스가 오픈클로보다 느리다. 단 좀 더 깊이 파고든다." — 속도는 오픈클로(Claude 4.8 탑재), 깊이는 에르메스(Codex 계열).
- **오픈클로 + 에르메스 코드 수집 활용**: "코드까지 확인해서 가져올 수 있는 거 다 가져오라고 하니까 이것저것 긁어오네요" — 멀티 에이전트를 정보 수집 용도로 병렬 활용하는 패턴.
