## 인사이트·노하우

- **Claude Code 주간 한도 소진 시 대응 순서** (P2 정리):
  1. OpenCode(무료) 또는 Commander Code 저렴 모델로 버티기
  2. Codex로 추가 작업
  3. MAX5 추가 결제
  - GAMSGO에서 1주일짜리 MAX×20 단기 구매도 가능 (P26)
  - Fable 5는 "지휘자 역할"로만 사용하고 토큰 남발 금지

- **Antigravity CLI 화면 깨짐 트러블슈팅** (Windows 환경):
  - 현상: 루프 중 토큰 소진 시 터미널이 이상한 문자로 가득 참
  - 원인: 토큰 다 써서 루프 진행 불가 → 한글 폰트 깨짐 의심
  - 해결 시도: `Ctrl+B`로 열었다 닫기 → 안 되면 세션 재시작

- **다중 AI 도구 병행 운용 패턴** (P18 실사례):
  - Cursor + Antigravity CLI + OpenCode 3개 병행
  - OpenCode(DeepSeek 무료) 80% 담당, 나머지는 Fable 전체 점검 시에만 사용
  - Antigravity CLI에 토큰 잔량 표시 기능 추가 업데이트 확인

- **Fable 5로 Unreal Engine MCP 제어** (P8 실사례):
  - MCP 브릿지를 통해 에디터 직접 조작 없이 작업 진행
  - 모션·판정 다듬기, 애니메이션 적용까지 가능
  - 블렌더도 MCP 연동으로 애니 스타일 배경 수정 툴 제작 사례 존재 (일본 개발자)

- **코딩 도구 가성비 비교** (P2 요약):
  - 가성비: Codex > Claude Code
  - 페이블 구독 기간 중 한시적으로 Claude Code 선호도 상승
  - 대규모 작업은 Codex, 일상 코딩은 Claude Code + Composer(Cursor) 병행
