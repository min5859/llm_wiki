# AI 요약

## 인사이트·노하우

- **에이전트 수보다 품질** — "에이전트 100개를 밤새 돌리는" 방식의 결과물은 처참하다. 핵심은 에이전트 한 명에게 "딱 떨어지는 일을 맡기고 깨끗하게 끝내는 것". "더 많이 띄우는 경쟁처럼 보이지만, 진짜 감각은 덜어내는 쪽에 있는지도 모른다."
- **에이전트 온보딩 비유** — "새 직원한테 매뉴얼 1,000장을 안기지 않는다. 당장 필요한 것만 알려주고 하나씩 더한다." → 에이전트 컨텍스트 설계에 동일하게 적용 가능.
- **삿갓(Satgat) 스킬 사용법** — 데모 페이지에서 API 키 설정이 필요. 무료 사용을 원하면 구글 제미나이 키(무료 사용량 있음)를 결제 계정 미연동 상태로 연동하면 된다. 로컬 안정적 사용은 레포 클론 권장.
- **링크 공유 팁** — 링크 앞뒤에 `]`, `[`, `(`, `{` 등을 붙이면 URL 파싱 오류 발생. "링크는 그냥 링크만 제공하는 것이 가장 깔끔하고 오류가 없다."
- **포트폴리오 기준** — AI 툴로 만든 프로토타입도 포트폴리오 활용 가능하나, 채용 담당자 시각에서 "단순히 만들었다" 수준은 서류 탈락 사유. 기본 기능 구현 이상이 필요.
