## 인사이트·노하우

- **Claude Tool Calling 오타 에러**: 컨텍스트가 꽉 찰 즈음 도구 호출 오타가 다발적으로 발생하는 현상이 보고됨. "Tool_Calling은 어느 에이전트든 다 에러가 있어서 세션 재시작해도 발생할 수 있다"는 것이 관리자 답변. 근본 해결책은 없으며 컨텍스트 관리가 우선.
- **Claude 먹통 트러블슈팅 순서**: 로그아웃 → 재부팅 순으로 시도. Claude 데스크톱은 무한재실행 에러, Claude Code는 작업 끊김 이슈가 있었으며, Claude 디자인 출시 시점에 업데이트가 하루 2번 뜨면서 뭔가 엉킨 것으로 의심됨.
- **Claude Code 터미널 사용 가능**: 기존 토큰 문제 해결 후 Claude Code 터미널에서도 사용 가능하게 됨. "클로드 디자인이 완전판이 되어가고 있다"는 평.

- **Sonnet / Haiku / Opus 용도 분리 실사용 팁** (P9 발언):
  - Opus: 전천후, 새로운 작업·판단이 필요한 경우
  - Sonnet: 이미 완성된 워크플로우 위에서 실행용
  - Haiku: 안 씀
