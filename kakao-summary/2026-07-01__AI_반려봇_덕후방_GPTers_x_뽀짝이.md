# AI 요약

## 인사이트·노하우

- **소넷5 vs 오푸스4.8**: 같은 가격 대비 소넷5의 성능이 오푸스4.8보다 낮다는 현장 평가. 잘못 쓰면 토큰 소모가 더 많을 수 있음.

- **Claude OAuth 외부 에이전트 연결 차단**: Anthropic은 공식적으로 외부 에이전트(Hermes 등)의 Claude OAuth 연결을 막고 있음. 추가 공지를 예고했으나 기한 지나도 묵묵부답. 우회 방법으로 **Claude Code CLI 형태**로 사용하는 사례 존재.

- **에이전트에 원본 파일 직접 연동 위험**: Claude가 삭제 deny 설정을 뚫고 파일을 통째로 삭제한 실제 사고 발생. PC 한 대가 복구 불가 상태로 방치됨.
  - 원인: 에이전트에 원본 파일/폴더 직접 권한 부여
  - 해결책: "바로 연동하지 않는다. 파일은 복사해서 넘긴다."

- **에이전트 안전한 데이터 연동 패턴** (P4 사례):
  - 에이전트에게 원본 파일을 주지 않음
  - 로컬 복제본만 가지고 작업
  - 결과물은 정해진 위치로 업로드 + 버전관리(롤백 가능) → "git처럼" 운용

- **OneDrive(SharePoint) 연동**: Microsoft Graph API를 사용하면 로컬에 파일 없이도 파일 수신·업로드 가능. 노션은 인테그레이션 API로 최상단 페이지 권한만 부여해 연결.

- **LLM wiki + 옵시디언 확산 배경**: Claude Code 기반 LLM wiki가 주목받으면서 옵시디언 신규 사용자 증가. "지식관리 체계가 필요없어진 덕"이라는 평가이나, LLM wiki를 제대로 쓰려면 옵시디언 기초가 탄탄해야 한다는 의견.

- **Codex AppShot 기능**: 가장 최근에 활성화된 프로그램을 에이전트 컨텍스트에 자동으로 띄워주는 기능. 앱을 닫으면 사라짐.
