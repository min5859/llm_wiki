## 인사이트·노하우

- **로컬 LLM 실용 모델 티어** (P2 경험담)
  - 9B 미만: 사실상 사용 불가 수준
  - 9B~: 무언가 할 수 있는 최소선
  - **31B**: 로컬 LLM 최대 가성비 — JSON 파싱이 제대로 되기 시작하는 분기점
  - 31B 구동 최소 RAM: **64GB** (P2·P32·P34 모두 64GB 환경)

- **모델별 사용 사례**
  - P34: Qwen3.6 (64GB)
  - P2: Qwen3 14B (태블릿, 미검증)
  - P32: Gemma4 (64GB, 간단한 크론 작업 자동화 테스트 중) — "아직은 답답한 것 같다"고 평가
  - Gemma4 툴콜링 실용성은 미검증 상태 (P96 질문에 답 없음 → 미해결 참조)

- **Hermes × Slack 모델 전환** (P97 질문 → P50·P11 응답)
  - 슬랙 스레드에서 Hermes 명령어가 동작하지 않는 현상 있음
  - 해결 방향: "Hermes에게 직접 바꾸라고 요청"하거나 "명령어를 수동으로 등록"해야 함 (텔레그램 대비 연동이 까다로운 구조)
