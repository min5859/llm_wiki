# AI 요약

## 인사이트·노하우

### AI 에이전트 전력 소비 실측 (KAIST 연구)
- AI 에이전트(다단계 계획·실행)는 질문 1건당 평균 **348.41Wh** 전력 소비
- 단순 챗봇 대비 최대 **136.5배** 전력, 응답시간 최대 **153.7배** 증가
- 하루 137억 건 요청 가정 시 데이터센터 수요 약 **198.9GW** 추정
- **시사점**: 업무 자동화에 에이전트 도입 전 "정말 다단계가 필요한 작업인가" 먼저 판단해야

### 기업 AI 비용 전략 변화: 토큰맥싱 → 모델맥싱
- 어려운 작업 → 고성능 모델(GPT-5.5, 클로드 페이블5), 단순·반복 → 저가/오픈소스 모델
- 모델 라우팅 플랫폼 도입 기업 비중: 작년 ~1% → 올해 5%
- **개인 적용 원칙**:
  - 저렴한 모델 가능: 요약, 번역, 단순 분류, 초안 정리
  - 고성능 모델 필요: 전략 판단, 긴 문서 구조화, 복잡한 코딩·검토
  - AI보다 직접 확인 필요: 법률·세무·의료·계약·중요 수치

### 클로드 페이블5 안전장치 부작용
- 7월 1일 재배포 시 탈옥 차단 안전 분류기 적용 (99% 이상 차단)
- 자체 벤치마크 결과: 디버깅 점수 86.2 → 25.9, 리팩토링 73.6 → 38.4 급락
- 원인: 모델 능력 저하가 아니라 **정상적인 코딩 요청까지 오퍼스 4.8로 우회 처리**된 영향
- 중요 작업에서 한 모델에만 의존하지 말고 차단·우회·응답 품질을 함께 확인해야

### 클로드 코드 보안팀 설득 사례
- 회사 보안팀이 git bash, Node.js, Python 설치 이유를 묻자 **"클로드의 손발"** 이라고 설명
- VSCode, 클로드 코드 익스텐션, Python까지만 허가된 사례도 존재
- node_modules 보안점검 소명 요구 사례도 공유됨

### 클로드 코드의 숨은 환경 감지 로직 논란
- 알리바바 7월 10일부터 직원 클로드 코드 사용 금지, 자체 도구 Qoder로 전환 지시 보도
- 중국 관련 프록시·시간대 식별 숨은 로직 의혹
- 앤트로픽 해명: **무단 리셀러·모델 증류 방지용** 실험적 조치(3월 도입), 곧 제거 예정
- **교훈**: AI 코딩 도구는 파일·셸 권한을 가지므로 "무엇을 읽고, 어디로 보내고, 어떤 환경 정보를 남기는지" 확인 습관 필요

### MVP vs 습작 개념 구분
- MVP(Minimum Viable Product): 부가 기능은 제외하되 **핵심 가치를 초기 사용자에게 전달**하면서 가설을 검증하는 버전
- 단순 기능 구현 연습은 MVP가 아니라 습작(프로토타입)에 가까움
- MVP의 목적은 완벽한 솔루션이 아니라 **"실제 문제를 해결하는가" 검증**

### Fable 시대의 병목 — 지시하는 사람
- 앤트로픽 클로드 코드 개발자 Thariq의 발언: "Fable은 내가 모르는 걸 얼마나 밝혀내느냐에 결과물이 발목 잡히는 첫 번째 모델"
- 모델이 아무리 좋아도 **지시 전 맥락 설계**가 결과물의 품질을 결정
- 답은 지시문 안이 아니라 **지시 이전**에 있음

허위조작정보 근절법, 7월 7일 전면 시행

7월 7일부터 개정 정보통신망법, 이른바 '허위조작정보 근절법'이 시행돼요. 이 법은 2025년 12월 24일 국회 본회의를 통과했고, 12월 30일 국무회의 의결을 거쳐 2026년 1월 6일 공식 공포됐어요.

■ 무엇이 달라지나

핵심 조항은 크게 세 가지예요.

1. 신고·처리 의무화
최근 3개월 하루 평균 이용자 수 100만 명 이상인 대형 플랫폼(네이버, 카카오, 구글, 유튜브 등)은 허위조작정보 신고접수 절차와 사업자 책임을 갖춰야 해요.

2. 가중손해배상제
손해를 끼칠 의도나 부당한 이익을 목적으로 허위조작정보를 유포한 언론사·유튜버 등에게 법원이 손해액의 최대 5배까지 배상액을 정할 수 있어요. 손해액 입증이 어려운 경우 법정 손해액 5000만원을 기준으로 산정돼요.

3. 반복 유통 과징금
법원 판결로 불법·허위조작정보임이 확정된 내용을 2회 이상 반복 유통하면 방송미디어통신위원회가 최대 10억 원의 과징금을 부과할 수 있어요.

대상은 조회수나 구독자 수가 대통령령 기준을 충족하는 '게재자'로, 구독자 10만 명 이상이거나 최근 3개월 게시물의 월평균 조회수가 10만 회를 넘는 경우부터 적용된다고 알려져 있어요. 대규모 플랫폼 사업자에게는 자율규제 정책 수립, 삭제 조치, 투명성 보고서 제출 의무도 함께 부과돼요.

■ 배경과 논란

이 법은 더불어민주당 언론개혁특별위원회가 2025년 10월 20일 발표한 '허위조작정보 근절안'에서 출발했어요. 발표 당시부터 야당과 언론단체는 "권력자에 대한 비판 보도를 위축시킬 수 있다"며 표현의 자유 침해 우려를 제기했어요.

법 통과 과정에서 고의성 없는 단순 오보까지 처벌 대상에 포함하려던 조항은 위헌 논란에 밀려 삭제됐고, 최종적으로는 '손해를 가할 의도'와 '부당 이익 목적'이 입증될 때만 처벌하는 방향으로 정리됐어요.

AI 생성 이미지와 딥페이크가 흔해진 지금, 이 법은 콘텐츠 제작자에게 출처 표시와 사실 확인의 부담을 키우는 동시에 플랫폼의 과잉 삭제 가능성이라는 새로운 쟁점도 함께 던지고 있어요.
