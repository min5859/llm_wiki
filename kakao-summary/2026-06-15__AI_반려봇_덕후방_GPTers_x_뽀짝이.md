## 인사이트·노하우

- **Claude `-p` 장시간 무응답 현상**: `✻ Newspapering… (1h 52m 8s)` 처럼 2시간 동안 응답 없는 행 업사태 발생(2026-06-15). 서버 과부하로 인한 일시적 장애였으며, 당일 오후 6시 20분경 복구됨.
  - 진단 사이트: [status.claude.com](https://status.claude.com/) 에서 현재 상태 확인 가능.

- **Claude MAX 요금제 + `-p` 크레딧 구조**: MAX 유저에게 대화형 한도 + API 크레딧(100달러 상당)을 함께 주는지 여부가 불분명. P13이 "본인 요금제만큼 API 크레딧 주는 것 같다"고 언급했으나, P3는 "아직 크레딧이 들어오지 않았다"고 했음 → 정책이 확정·반영되지 않은 과도기 상태로 보임.

- **Fable5 프롬프트 주입 방법**: `claude --append-system-prompt` 옵션에 공개된 Fable5 프롬프트를 직접 입력하는 방법이 있음 (P113 제보).

- **OpenClaw + Codex GPT-5.4 → 5.5 업그레이드 안내**: GPT-5.4 사용 중 5.5로 업그레이드하라는 메시지가 표시됨. 5.4가 즉시 차단된 것은 아닐 가능성이 높으나 공식 발표 확인 필요.

- **뽀짝이(카카오 채팅 봇) 구현 한계**: "뽀짝이"라는 이름 감지 시 최근 10개 대화를 읽고 응답하는 구조. Codex 인증이 자주 풀려 불안정. 세션 유지가 안 되는 문제 있음. 오픈채팅 외 일반 그룹톡에도 초대 가능하나 동일한 불안정 문제 존재.

- **OCI(Oracle Cloud Infrastructure) 무료 티어 축소 제보**: 기존 4 CPU / 24GB에서 2 CPU / 12GB로 줄어든 것으로 보인다는 제보 (P48, 미확인 — 직접 확인 권장).
