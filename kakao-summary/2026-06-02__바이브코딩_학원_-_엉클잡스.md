# AI 요약

## 인사이트·노하우

### 멀티 AI 툴 컨텍스트 관리 워크플로 (P25 공유)
복수의 AI 도구(Claude, Codex, Gemini 등)를 한 프로젝트에 혼용할 때의 실전 방식:

- **역할 분담**
  - Anthropic CLI(클코) → 플래닝 + 전체 뼈대 (1M 컨텍스트 활용)
  - Codex → 코드 작성 / 코드 리뷰
  - Anti CLI → 디자인 / 간단한 작업

- **컨텍스트 인계 방법**
  - 작업 시작 전 구획(모듈/기능 단위)을 미리 분리
  - `handoff` 문서를 만들어 AI 간 컨텍스트 전달
  - 스킬/지식은 공유 가능한 형태로 유지

- **설정 파일 전략**
  - `AGENTS.md` 를 공통 기준점으로 설정
  - 도구별 설정: `CLAUDE.md`, `GEMINI.md` 등 분리 작성

### Claude 주간 토큰 한도 리셋 방법
- Claude 토큰 한도 초과 시 **Reddit에 사유를 잘 작성해서 요청**하면 리셋해주기도 함 (복불복)

### Cursor에서 Grok Builder 사용 시
- Cursor에서 Grok Builder를 사용하면 **Composer 2.5와 쌍으로 동작**
- Grok CLI는 속도는 빠르나 얼리 베타 단계로 성능은 아직 미흡하다는 평
