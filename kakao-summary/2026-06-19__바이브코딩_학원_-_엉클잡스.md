## 인사이트·노하우

### AI 도구 내장화 트렌드 (2026-06-19 기준)
- **Adobe** Photoshop·Premiere·Illustrator·InDesign·Frame.io에 AI Assistant 공개 베타 도입. 자연어 지시로 배경 교체, 리사이즈, 클립 정리, 타임라인 마커 등 반복 작업 처리 가능. "편집 기술"보다 "원하는 결과를 잘 지시하고 검수하는 능력"이 더 중요해지는 흐름.
- **Google Ad Manager**에 Gemini 기반 Ask Ad Manager 베타 공개. 광고 라인아이템 문제 진단, 리포트 생성, 설정 화면 이동을 대화형으로 처리. 실제 계정 데이터 기반이므로 권한 설정과 결과 검수 필수.
- **GitHub Copilot** code review가 저장소 루트의 `AGENTS.md` 파일을 읽고 프로젝트 규칙을 반영한 리뷰 가능. MAI-Code-1-Flash 모델을 CLI·앱·JetBrains·Xcode 등 다수 환경으로 확대.

### AGENTS.md 실용 팁
- 저장소 루트에 `AGENTS.md` 작성 → Copilot 리뷰 품질 향상.
- 담을 내용 예시: 말투, 금지할 작업, 검수 기준, 파일명 규칙, 사람 승인이 필요한 행동 (5가지 룰 형태).

### AI 보안 사고 교훈 — "모두의 창업" 개인정보 유출 사례
- 서비스 전체 회원 데이터가 인증 없이 노출되는 수준의 결함 발생.
- 원인: **authGuard 미적용** — "해당 요청을 하는 사람이 이 응답값을 볼 자격이 있는가"를 검증하는 로직 부재.
- 올바른 설계: `authGuard`로 요청자 자격 검증 + 관리자는 `adminGuard`로 별도 권한 체계(조회/수정/삭제 분리) 적용.
- 관련 발언: "하네스 엔지니어링에도 Guardrails라는 개념이 있는 판국에…" — AI로 빠르게 만들더라도 권한 체계 설계는 생략 불가.
- 교훈: 바이브코딩으로 빠르게 만들어도 인증·인가 체계는 AI에게 명시적으로 요청하거나 직접 검수해야 함. "딸깍하면 되지 않냐"의 반례로 인용 가능한 사례.

### Codex 신기능 — Record & Replay
- 화면을 녹화하면 Codex가 분석하는 **Record & Replay** 기능 탑재 예정/공개.
- 토큰 리셋 외에도 지속 기능 확장 중.

### Claude Code 아티팩트 변화
- 기존: Claude 웹/데스크탑에서만 인터랙티브 차트·보고자료 확인 가능.
- 변화: Claude Code에 내장되면서 **협업 툴로 전환** — 팀 단위 공유·검토 용도로 활용 가능해짐.

### mole + Codex/Claude Code 조합 — 맥 용량 정리
- `mole` + Codex 또는 Claude Code 조합으로 macOS 불필요 파일 대량 정리 가능. 실사용 사례: 400GB 회수.
