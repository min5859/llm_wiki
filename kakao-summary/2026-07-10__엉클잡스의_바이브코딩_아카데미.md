## 인사이트·노하우

- **ChatGPT Work 도입 시 자동화 4단계 분리 권고**: "자료 수집 → 초안 작성 → 사람이 승인 → 발송" 순서로 나눠 첫 자동화 흐름을 구성할 것. 에이전트 실수 위험을 줄이는 실용 패턴.
- **AI 에이전트 시대의 업무 설계 원칙**: "질문 잘하기"보다 "일을 맡길 단위로 쪼개기"가 더 중요. 월말 정산·제안서 초안·리서치 정리처럼 결과물이 분명한 일부터 위임 실험.

- **코덱스 + 클로드 동시 작업 가능**: "서로 번갈아가면서 작업하라 하거나 둘이 논의해서 업무 범위 조정하라 하면 알아서 잘해준다"는 실사용 경험 공유.

## 자동화·시스템 적용 후보

- **의류 이미지 → JSON 키워드 DB 자동화 파이프라인**: 의류 사진을 AI에 넣고 아래 구조의 JSON을 뽑아 키워드 DB로 축적하는 워크플로. 어떤 LLM이든 적용 가능(Gemini, Claude, Grok, Codex).

```json
{
  "top": {
    "type": "...",
    "color": "...",
    "fit": "...",
    "fabric": "...",
    "neckline": "...",
    "sleeve_length": "...",
    "length_and_tuck": "...",
    "layering": "...",
    "style": "...",
    "details": ["...", "..."]
  },
  "bottom": { "type": "...", "color": "...", "fit": "...", "fabric": "...", "length": "...", "style": "..." },
  "legwear": { "type": "...", "tone": "...", "appearance": "..." },
  "shoes": { "type": "...", "style": "...", "color": "...", "design": "..." },
  "accessories": { "items": ["..."], "avoid": ["..."] },
  "overall_style": ["...", "..."]
}
```

  활용법: 결과물을 쌓아 "어떤 키워드가 어떤 재질을 만들었는지" 역추적 → 이미지 생성 프롬프트 정밀도 향상.

- **멀티 에이전트 동시 작업 패턴**: 코덱스 + 클로드를 병렬로 돌릴 때 "번갈아가며 작업" 또는 "업무 범위를 서로 논의해서 조정"하는 방식으로 충돌 없이 운용 가능.
