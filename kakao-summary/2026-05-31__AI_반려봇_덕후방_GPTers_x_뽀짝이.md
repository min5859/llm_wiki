# AI 요약

## 인사이트·노하우

- **Hermes Agent 입력 토큰 14% 절약 업데이트**
  - 파일 읽기 작업 시 입력 토큰을 평균 14% 절약하는 방법이 메인 브랜치에 적용됨.
  - 적용 방법: `hermes update` 실행.
  - 추가로, MCP/플러그인 도구가 컨텍스트 창의 **10% 이상**을 차지할 경우 자동으로 "임의 도구 로드" 기능이 활성화됨 → 컨텍스트 타협 없이 모든 도구 접근 가능.

- **소형 로컬 모델(4B)에 외부 도구(브라우저 등) 부여 시 보안 위험**
  - Gemma3:4b를 서브에이전트로 사용 시 보안 체크 결과: "작은 모델에 외부세계 보는 도구를 주지 마세요. 특히 브라우저는 칼입니다. 4B한테 쥐여주면 손가락 날아갑니다."
  - 실용적 한계도 겸비: 컨텍스트 윈도우가 작아 원하는 결과가 잘 안 나오고, 응답 속도도 느림.
  - 로컬 Qwen은 도구 호출이 불안정, Gemma4는 실행 불가 → 로컬 소형 모델로 에이전트 구성 시 현실적 제약 큼.
