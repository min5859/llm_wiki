## 인사이트·노하우

- **Cursor Composer 2.5 모델 구조**: Cursor Composer는 자체 모델이 아니라 "키미(Kimi)를 증류하고, 그록(Grok) Colossus 컴퓨팅으로 강화학습"한 모델. 경쟁 모델 대비 가성비가 높다는 평가.
- **Cursor 타 모델 사용 시 주의**: Composer 제외 타 모델 사용 시 사용량이 빠르게 소모됨. 현재 기준으로는 "킹덱스(Codex)" 조합이 최고라는 의견.
- **Cursor 75% 할인 복귀 오퍼**: `https://cursor.com/activate/welcome-back/offer` — 기존 유료 사용자 대상 복귀 할인. 신규·무료 전용자는 해당 없음. 단, 신규/무료 사용자 중 바로 구독된 사례도 있어 "복불복" 가능성 있음.
- **Cursor 구독 취소 시 50% 할인 제안**: 취소 시도 시 다음 달 50% 할인 오퍼가 나온 사례 있음.
- **Cursor 학생 할인**: edu 도메인 이메일 보유 시 학생 할인 적용 가능.
- **Claude Code Setup 플러그인**: 프로젝트를 자동 스캔해 Hooks·스킬·MCP 서비스·서브 에이전트·자동화 설정을 원클릭으로 추천·완료해주는 플러그인. 설치 명령: `/plugin install claude-code-setup@claude-plugins-official`
- **에이전트 활용 마인드셋**: "에이전트를 쓰는 건 코딩이 아니라 일을 잘 시키는 것. 전략 게임(스타크래프트의 프로브/SCV에 일 시키기)처럼 접근해야 한다."
- **Zed + 에이전트 기능**: Zed는 가볍지만 기본적으로 컴퓨터 직접 제어·웹브라우저 접근 불가. 유료 구독 시 에이전트 기능 사용 가능.
- **VS Code + Kilo 조합**: VS Code에 Kilo 확장 설치 후 OpenRouter 또는 타 API 키 연결하면 가벼운 에이전트 환경 구성 가능. Cursor·AntiGravity 대비 경량 대안.
