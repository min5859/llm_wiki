## 인사이트·노하우

- **Hermes + Claude 채널 직연결 개인 메신저 에이전트**: 카카오톡·텔레그램·디스코드·슬랙 등 제3자 서비스를 경유하지 않고 Hermes 에이전트와 Claude 채널 에이전트를 직접 연결하는 구조. 문서를 외부 서비스에 업로드하지 않아도 되므로 기업 문서 유출 리스크 차단 가능. 완전한 프라이버시가 필요하면 로컬 LLM과 연동하면 됨.

- **Claude Fable 구독 크레딧 소진 전략**: 주간 한도 리셋 시간은 계정마다 다름(요일·시간 모두 상이). "항상 아껴쓰다가 초기화 다가오면 막쓰기"가 실질적 운용 패턴으로 확인됨. Fable 구독 모델 사용 허용 종료는 한국 기준 2026-07-08 16:00(정확히 15:59).

- **Codex 초기화권(리셋권) 동작**: 사용량 0%가 되어 채팅 불가 상태가 된 뒤 수동으로 초기화권을 사용하면 100%로 복구되는 방식. 자동 소진되지 않음. OpenClaw가 설치된 기기에 Codex 앱을 추가 설치하면 초기화권이 중복 표시(4개)되는 케이스 보고됨.

- **Fable vs Opus 역체감**: Fable 크레딧을 모두 소진한 후 Opus로 전환했을 때 성능 역체감이 크다는 사용자 경험 보고. Claude Code는 자동으로 Opus로 전환되는 것으로 추정.

- **Fable API 비용**: 구독제 대비 API 사용 시 체감 비용이 "20배"에 달한다는 평가.
