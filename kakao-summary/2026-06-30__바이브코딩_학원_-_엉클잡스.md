### Codex 한도 리셋 잔여량 확인 프롬프트

Codex 구독 한도 초기화 시점이 궁금할 때, 아래 프롬프트를 Codex에 그대로 붙여넣으면 로컬 시간 기준으로 만료일을 알려준다.

```
본 기기 Codex 자격 증명을 사용하여 rate-limit reset credits를 확인해 주세요:

~/.codex/auth.json 파일에서 tokens.access_token을 읽어, 다음 인터페이스에 요청하세요:
(https://chatgpt.com/backend-api/wham/rate-limit-reset-credits)

요구사항:
1. access_token, refresh_token, cookie 또는 전체 고유 ID를 출력하지 마세요
2. available_count와 각 credit의 status/title/granted_at/expires_at만 요약하세요
3. granted_at/expires_at를 UTC에서 로컬 시간으로 변환하세요
4. 상태 코드가 401로 반환되면 자격 증명 만료 또는 올바른 Authorization 헤더가 누락된 것입니다
```

- 401 반환 시 → 자격 증명 만료 또는 Authorization 헤더 누락
- 엔드포인트: `https://chatgpt.com/backend-api/wham/rate-limit-reset-credits`

### 블로그 검색엔진 색인 자동화 파이프라인

Claude Code로 블로그 글 발행 시 검색엔진 색인 요청 자동화를 구현한 사례. 5개 검색엔진 결과:

| 검색엔진 | 자동화 가능 여부 |
|---|---|
| Bing | ✅ IndexNow 한 방으로 처리 |
| 네이버 | ✅ IndexNow에 포함됨 (핵심) |
| Yandex / Seznam | ✅ IndexNow 동시 통보 |
| 다음 | ⚠️ 공식 API 없어 반자동만 가능 |
| 구글 | ❌ 일반 글은 자동화 불가 → 사이트맵+수동 |

**주의**: HTTP 200 응답이 떠도 실제 실패일 수 있음 → 응답 본문을 반드시 확인해야 함.

### Apple Silicon 맥 구매 기준: 코딩 용도

참여자들의 합의:
- **RAM 우선** — CPU 코어 수보다 RAM이 체감 성능에 더 큰 영향
- **32GB가 마지노선** — 48GB 사용자도 메모리 압력 경고 없이 사용 중
- CG/렌더링 병행 시 24GB는 일상에서도 부족한 경우 多
- SSD 용량도 중요 — 1TB는 실사용 시 압박 있음
