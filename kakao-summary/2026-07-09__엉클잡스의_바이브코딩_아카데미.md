# AI 요약

## 인사이트·노하우

- **Claude Code CLI에서 다중 계정 전환** — 데스크탑 앱은 계정 전환 시 세션이 튕기지만, CLI(`claude`명령)는 세션이 열린 상태에서 `/login`으로 계정을 전환해도 기존 세션이 유지된다. 다른 계정으로 넘어가 작업을 그대로 이어받는 것도 가능.
- **Fable(Claude)으로 웹앱 제작 시 토큰 비용** — 30분 에이전트 실행에 프로젝트 1개 기준 약 $88 소진 사례가 보고됨. 크레딧 충전보다 구독 추가가 단가 면에서 유리하다.
- **Codex 컨텍스트 관리** — Codex는 컨텍스트가 차면 자체 압축(compact)하며 "새 세션 불필요"라고 주장하지만, Claude는 새 세션을 권고하는 방식. 실제로는 사용자가 직접 새 세션을 여는 것이 안전.
- **Codex에서 이미지 생성** — Imagen 2.0으로 이미지를 뽑으려면 `imagen` 스킬을 사용. Codex Spark로 프롬프트를 생성한 뒤 GPT로 처리하면 사용량이 각각 다르게 차감되어 효율적.
- **Claude Effort·Model 설정** — Claude Code 사용자라면 반드시 확인해야 할 설정. 엉클잡스 Threads 포스트에 정리됨(아래 링크 참조).
