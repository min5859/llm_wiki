# AI 요약

## 인사이트·노하우

- **GLM-5.2 로컬 구동 VRAM 요구사항**: 4bit 양자화 기준 500GB+ 이상, KV 캐시·컨텍스트 윈도우까지 고려하면 700~800GB 수준 필요. 개인이 구동하기는 사실상 불가. 단, **256GB 통합 메모리 Mac + MoE 오프로딩** 조합이면 1×24GB GPU + 256GB RAM 환경에서 작동 가능.

- **AI 봇 무한 루프 사고 패턴**: 봇이 "보고해주세요~" 같은 루프에 빠져 일간 사용량을 5시간 만에 소진한 사례. GLM 세팅 오류로 악의적 반복 요청 판정·리밋을 먹은 사례도 있음. → **세팅(종료 조건·루프 가드) 사전 검증 필수**.

- **Codex 사용량 증가 체감**: 같은 작업 기준 1~2주 전보다 소진 속도가 빨라졌다는 제보. 원인 불명.

- **step3.7 flash(Stepfun) 활용 후기**: '관제'와 '리뷰' 역할에서 탁월, 응답 속도도 빠름. 메인 모델로 쓰기엔 아직 부족하지만 서브 보완재로 유용. GPT-5.5가 경로 이탈·과도한 실행 의지로 관제/리뷰에서 빵꾸를 낸다면 서브 모델로 보완하는 방식이 효과적.

- **Claude Slack 연동 시 레포 1개 제한**: 슬랙에 Claude 앱을 붙이면 레포지토리를 1개만 지정 가능 → 멀티 레포 환경에서 약간 애매. (비즈니스 계정은 월간 사용량만 표시되고 단시간 집중 사용 시 더 빨리 막힌다는 추가 제보 있음)

- **일레븐랩스 보이스 클론 팁**:
  - **Instant Voice Clone**: 30초 녹음으로 클론 생성 가능. 단 다양한 감정을 담아 녹음해야 다용도로 활용 가능. 녹음 내용과 유사한 음성만 재현되는 한계가 있음.
  - **Voice Design**: 텍스트 프롬프트(예: "우락부락한 근육질 거구 남자 아저씨")만으로 목소리 생성 가능.
  - **생성 음성 파일 별도 보관 권장**: 일레븐랩스로 생성한 음성을 모아두면, 나중에 더 저렴한 별도 보이스 모델에 재학습시켜 비용 절감 가능.
  - **노래 모델 생성 조건**: 다양한 피치의 노래 음성 소스 15분+ 이상, 5시간+ 학습 필요. 수고스러움.
  - **fish audio** GPU 위에 올려 본인 목소리를 거의 완전 복사해 쓰는 방식도 있음.

- **Slack `/명령어` 등록**: Slack CLI로 `/명령어` 등록 요청을 하면 처리 가능하다는 팁.
