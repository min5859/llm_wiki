## 추천·자료

- **Google I/O 2026 Puzzle Play** — I/O 행사 연계 미니게임으로, Hole in One, Nonogram, Word Wheel, Supersonic Bot, Stretchy Cat 등 5종 퍼즐이 있다는 맥락에서 공유됨. [Google I/O 2026 Puzzle](https://io.google/2026/puzzle/play/)

- **SIE (Superlinked Inference Engine)** — Superlinked 측 추론 엔진(저장소/검색·추론 스택 일부로 이해하면 됨)으로, 위 인수 소식과 함께 링크됨. [github.com/superlinked/sie](https://github.com/superlinked/sie)
- **MYCL 콘테스트 + Hermes Agent + Grok** — 에이전트·생성 설정(이미지·영상 Generator) 매뉴얼이 별도 노트로 정리되어 함께 언급됨. [콘테스트](https://mycl.io/contest), [매뉴얼 노트](https://share.note.sx/27fyorgp)
- **Learn Harness Engineering (한국어)** — 에이전트/툴 **하네스** 설계·엔지니어링 학습 자료로 제시됨. [walkinglabs 문서 (KO)](https://walkinglabs.github.io/learn-harness-engineering/ko/)
- **Free Coding Agent (FreeBuff)** — 코딩 에이전트를 무료로 써보려는 목적의 도구로 소개됨. [freebuff.com](https://freebuff.com/)
- **「멀티 에이전트 노하우」영상 (단테랩스)** — 멀티 에이전트 운용 노하우를 담은 유튜브로 공유됨. [YouTube](https://youtu.be/Z9NDvIdMi9E)
- **인프런 「혼자 공부하는 바이브 코딩 with Claude Code」(무료)** — Claude Code 기반 바이브 코딩 입문 강의로 링크됨. [인프런 강의](https://www.inflearn.com/course/바이브-코딩-입문-강의-혼자-공부하는?cid=340120)
- **네이버 뉴스 기사** — 구체적 주제는 입력에 없음(불명). [기사](https://n.news.naver.com/mnews/article/003/0013953367)
