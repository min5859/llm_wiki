## 인사이트·노하우

- **Cursor Composer 2.5 모델 구조**: Cursor Composer는 자체 모델이며, Kimi를 증류(distillation)하고 Grok의 Colossus 컴퓨팅 인프라를 활용해 강화학습을 진행한 것으로 알려짐. Grok 모델 자체를 쓰는 것은 아님.
- **Cursor 타 모델 사용 시 주의**: Cursor에서 Composer 외 타 모델을 사용하면 사용량이 빠르게 소진됨. "커서는 컴포저 제외하고 타모델쓰면 사용량 쭉쭉 달아서 주의 요망".
- **Cursor 학생 할인**: `.edu` 도메인 이메일 계정 보유 시 학생 할인 적용 가능.
- **Cursor 구독 취소 시 50% 할인 제안**: 취소 시도 시 다음 달 50% 할인을 제안받은 사례 있음.
- **Cursor Composer 2.5 할인 조건**: 복귀 유저 대상 75% 할인은 기존 유료 이용 이력이 있어야 적용됨. 신규·무료 사용자는 일부 바로 구독 가능한 경우도 있음(랜덤 또는 조건 불명).
- **Claude Code Setup 플러그인**: 프로젝트를 자동 스캔해 Hooks, 스킬, MCP 서비스, 서브 에이전트, 자동화 설정을 원클릭으로 추천·설정. 설치 명령: `/plugin install claude-code-setup@claude-plugins-official`
- **Zed 에디터 특성**: 가볍지만 인터페이스가 VS Code·Antigravity와 달라 적응 필요. 유료 구독 시 에이전트 기능 사용 가능. 로컬 LLM(예: DeepSeek)을 OpenRouter로 연결하면 컴퓨터·브라우저 직접 접근 불가.
- **Zed에서 에이전트 기능 대안**: VS Code에 Kilo 플러그인 설치 후 OpenRouter 또는 타 API 키 연결하는 방법이 있음.
