## 인사이트·노하우

- **DeepSeek / Kimi를 IDE에서 사용하는 방법**
  - 커서(Cursor)나 Antigravity IDE에서 직접 지원하지 않는 경우:
    - Kimi: 직접 구독 후 CLI로 사용 가능
    - DeepSeek: API Router(OpenRouter 등) 경유 필요
    - 대안 IDE로 **Zed** 강력 추천 ("가볍다, 강추")
  - Zed 설정 관련 공개 포스팅은 2026-05-25 기준 아직 없음 → 독학 필요

- **DeepSeek API 비용 최소화 전략**
  - Command Code 1달러 구독 → DeepSeek 10달러치 사용 가능
  - OpenCode Go 첫 달 5달러 요금제로 찍먹
  - OpenRouter 충전 후 직접 사용 (코딩 경험 있을 때 추천)
  - 실사용 후기: "100달러 충전 후 한 달에 8달러 사용" — 소모가 매우 적음
  - 활용 팁: "죽어있는 웹앱에 DeepSeek API 붙이면 좋다" → VPS + 도메인(가비아) + 앱 + DeepSeek API 조합

- **작업 깊이에 따른 모델 선택 가이드** (실제 사용자 패턴)
  - 가벼운 작업: Command Code → OpenCode Go
  - 딥한 작업: Claude 또는 GPT
  - 가성비 최고: Cursor Composer 2.5 (2026-05-25 기준 커뮤니티 평이 좋음, "한때 이기는 하지만")

- **DeepSeek V4 Flash 무료 제공**
  - Nous Portal에서 무료로 재오픈됨 (Hermes Agent 사용자 참고)
  - 링크: [portal.nousresearch.com](https://portal.nousresearch.com/manage-subscription)
